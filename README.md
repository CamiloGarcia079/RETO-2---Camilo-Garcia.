# Programa de Notas de Alumnos

## Descripción
Este proyecto consiste en una página web interactiva para registrar y gestionar las notas de tres certámenes de 10 alumnos. El sistema permite ingresar nombres y calificaciones, calcular promedios individuales, obtener estadísticas generales del curso, identificar alumnos aprobados y reprobados, y mostrar un ranking ordenado por promedio final.

## Uso
1. Ingresa el nombre del alumno en el formulario.
2. Escribe las tres notas de los certámenes.
3. Presiona el botón `Agregar alumno`.
4. Repite el proceso hasta completar los 10 alumnos.
5. Revisa la sección de resultados para ver:
   - Las notas y promedio de cada alumno.
   - Los promedios del curso por certamen.
   - El promedio general del curso.
   - La cantidad de aprobados y reprobados.
   - El ranking de alumnos ordenados por promedio.
